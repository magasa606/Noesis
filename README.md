# Noesis

**Noesis** is my personal portfolio documenting my journey into **Data Analytics and Business Intelligence**, where data is explored not only as numbers but as a source of structured knowledge and decision support.

This repository contains selected projects developed during my learning journey in analytics, statistics, and business intelligence. Each project focuses on transforming raw data into **insightful analysis, interpretable models, and actionable business recommendations**.

My academic background is in philosophy, which shaped my interest in how knowledge is constructed. In the context of data analytics, this translates into transforming **data → evidence → insight → decision**.

## 14. Workforce Retention Strategy (Final BI Project)

Business Intelligence project analyzing **employee attrition using IBM HR Analytics data**.

Key components:
- Data preprocessing
- Exploratory Data Analysis
- Feature selection
- Logistic Regression modeling
- Tableau dashboard visualization

Key insight:
Employees working overtime show nearly **2× higher attrition risk**, while seniority and income stability correlate with higher retention.

The predictive model achieved an **AUC score of 0.79**, indicating strong discrimination ability.

Focus:
Supporting **human capital strategy and workforce retention decisions**.
